# Objective:
Build and boot a custom Linux kernel for ARM using kw. The customization will be a different name to the kernel image. The name I picked was teste-tutorial2.

# Problems:
This part went smoothly, probably because kw already automates the majority of the process.

# Results:
```bash
root@localhost:~# uname --kernel-release
7.0.0-rc1teste-tutorial2
```

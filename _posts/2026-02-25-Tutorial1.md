# Objective:
Set up a test environment for Linux Kernel devlopment using QEMU and libvirt.

# Problems:
* The link for the Debian image was wrong: 
```bash
wget --directory-prefix="${VM_DIR}" http://cdimage.debian.org/cdimage/cloud/bookworm/daily/20250217-2026/debian-12-nocloud-arm64-daily-20250217-2026.qcow2
```

    So I had to manually go to "https://cdimage.debian.org/cdimage/cloud/bookworm/daily/" and download it.
